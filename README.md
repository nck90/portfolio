## INSTALL
아래 명령어를 사용해 프로젝트를 클론하고 의존성을 설치한 뒤, 로컬에서 실행할 수 있습니다.
> [Node.js](https://nodejs.org/ko/)가 필수로 설치되어 있어야 합니다.

```bash
npm install
```
```bash
npm run dev
```
위 명령어 실행 후, [localhost:3000](http://localhost:3000)에서 애플리케이션을 확인할 수 있습니다.
